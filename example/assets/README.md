# Sample Assets for Fluvie Examples

This directory contains sample media files used in the Fluvie example gallery.

## Directory Structure

- `videos/` - Sample video files (MP4 format recommended)
- `images/` - Sample image files (PNG, JPG format)
- `audio/` - Sample audio files (MP3, WAV format)

## Recommended Sources for Free Assets

### Videos
- **Pexels**: https://www.pexels.com/videos/ (Free stock videos)
- **Pixabay**: https://pixabay.com/videos/ (Free videos)

### Images
- **Unsplash**: https://unsplash.com/ (Free high-quality photos)
- **Pexels**: https://www.pexels.com/photos/ (Free stock photos)
- **Pixabay**: https://pixabay.com/images/ (Free images)

### Audio
- **Freesound**: https://freesound.org/ (Free sound effects and music)
- **Pixabay Music**: https://pixabay.com/music/ (Free music tracks)
